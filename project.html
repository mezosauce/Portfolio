WAR

Implemented the classic card game "WAR" using fundamental data structures. I designed a personal deck that operated as a Queue to manage the order of cards drawn during gameplay, and a side pile that functioned as a Stack to track dynamically accumulated cards. This approach allowed me to effectively model real-time changes in game state using appropriate abstract data types.
